# Murdoku Studio

Editor y reproductor de Murdoku — acertijos de lógica y asesinatos. Hecho con HTML, CSS y JavaScript vanilla (sin frameworks ni bundlers).

## Cómo usar

Abrí `index.html` en tu navegador. No requiere build ni servidor.

- **Jugar**: seleccioná un caso, arrastrá sospechosos al tablero, verificá la solución.
- **Editor**: creá tus propios casos con el panel editor. Definí zonas, objetos, sospechosos y pistas.
- **Generar**: el botón "Generar" crea un caso procedural con zonas, víctima y asesino en una misma zona.

## Tests

```bash
npm install
npm test
```
